# Ejecución

**Prerequisitos:**  Node.js

1. Instalar dependencias:
   `npm install`
2. Ejecutar la app:
   `npm run dev`

## Notas

**Colores base**
Los colores base de cada facción:
- amazons    : rgb(211, 205, 58)
- barbarians : rgb(44, 124, 182)
- dwarves    : rgb(145, 65, 187)
- daemons    : rgb(189, 46, 46)
- elves      : rgb(67, 157, 65)
- orcs       : rgb(208, 105, 28)
- undead     : rgb(58, 58, 58)
- mercenaries: rgb(89, 58, 2)
- inferno    : rgb(117, 8, 8)
- titans     : rgb(239, 239, 239)

**Tamaño de los bloques**
Tamaños originales:
- Bloques Pequeños (Small Blocks): Aproximadamente 16mm (ancho/largo) x 8mm (alto).
- Bloques Medianos (Medium Blocks): Aproximadamente 21mm (ancho/largo) x 8mm (alto).
- Bloques Rectangulares (Rectangular Blocks): Alrededor de 21mm x 26mm (ancho x largo) x 8mm (alto). 

Tamaños nuevos (al utilizar solo un bloque por hexágono, estos pueden ser más grandes y vistosos):
- Light: 23mm x 23mm
- Medium: 27mm x 27mm
- Heavy: 31mm x 31mm
- Elite: 27mm x 23mm
- Structure: 31mm x 31mm

**Tamaño de los stickers**
El tamaño de los stickets, puede ser 1mm más pequeño que el de los bloques:
- Light: 22mm x 22mm
- Medium: 26mm x 26mm
- Heavy: 30mm x 30mm
- Elite: 26mm x 22mm
- Structure: 30mm x 30mm
